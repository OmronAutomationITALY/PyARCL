Class to be used to generate a Fleet Dashboard.

Class used to generate a Fleet Dashboard. The dashboard will contain three matplotlib.plt plots, one being a pie chart of the robot states, and two being a real-time plots of the robot states and job queue.

FM

The FleetManager object that hold the robot to which statistics will be calculated

Type

FleetManager

delta_t

The time interval in seconds at which refresh the data and update the plots. Default: 20 [s]

Type

int

reset_trip

Parameter that controls whether or not to run the (resetTrip()) method before starting to plot the data.
